Приложение основано на покупке лицензионных фильмов. 
Каждый пользователь может зарегестрироваться и войти свой аккаунт.
Каждый пользователь покупает фильм, и купленный фильм переходит к пользователю в библиотеку в раздел - "Favorite".
Каждый пользователь через поиск может найти необходимый фильм.
Все данные о пользователях и купленных фильмах хранятся на Google Firebase.
Как приходит оповещение ответственному за приложение, что пользователь приобрёл фильм, в течении некоторого времени с пользователем связываются, и отправляют фильм на электронную почту.
